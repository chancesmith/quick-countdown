# Quick Countdown
Add countdown by URL variables

Check it out
https://chancesmith.github.io/quick-countdown/

## Countdown variables
- `?days=4`
- `?hours=2`
- `?mins=7`

## Extras
- `?title=Countdown until Javascript Meetup`
- `?message=Countdown complete!`
